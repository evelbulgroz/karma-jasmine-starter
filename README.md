# karma-jasmine-starter
A simple starter project for running [Jasmine](https://jasmine.github.io) tests with [Karma](https://karma-runner.github.io/3.0/).

## Prerequisites
To use this project, make sure you first have the following (or better) installed:
* [NodeJS](https://nodejs.org/) v8.11.4
* [NPM](https://www.npmjs.com/) v6.4.0

You will also need a terminal app. Windoze and and Linux variants are both fine.

Lastly, make sure your Google Chrome install binary is added to your environment variables. E.g. on Windoze, add `C:\Program Files (x86)\Google\Chrome\Application\chrome.exe` to your `PATH`.


## Install
Copy or clone from repo, then run `npm install`.

## Run
Run `npm test`.

## Use
Add `.js` source and spec/test files to the `js` directory, or a subfolder thereof.

Suggested file naming convention (see provided example):
* source: `somename.js`
* test: `somename.spec.js`
